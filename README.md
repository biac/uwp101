## プロジェクト名

書籍のどの部分のサンプルコードなのかを分かりやすくするため、プロジェクト名を書籍とは変えています。 
名前の先頭に 「\_」 で区切って章・節の番号を付けました。

**例** 
08_6_ImageList: 8章6節で完成させた 「ImageList」 プロジェクト
